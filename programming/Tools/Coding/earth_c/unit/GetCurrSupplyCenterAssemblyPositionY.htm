int GetCurrSupplyCenterAssemblyPositionY()

Scope: private

Parameters
none

Return values
Y-coordinate of Supply Center's assembly point.

Description
Becouse assembly point is calculated by function GetCurrSupplyCenterAssemblyPositionX and stored in static variable so first you should call function GetCurrSupplyCenterAssemblyPositionX and then GetCurrSupplyCenterAssemblyPositionY and GetCurrSupplyCenterAssemblyPositionZ.
