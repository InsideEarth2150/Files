EarthC language description

EarthC is a relatively simply language heavilly base on C++. We use it to control game's objects, modify AI behaviors, create missions and many many more....

The basic structure of EarthC class

BaseClassIdentifier "Script name"
{
    consts definitions
    member definitions
    functions definitions
    states definitions
    events definitions
    commans definitions
}

"Script name" is a name which this script has in game
