void CallUpgrade(unit uTarget)

Scope: private

Parameters
uTarget - object which we have to upgrade.

Return values
none

Description
This function begins upgrading of object uTarget, only if we can upgrade something in object uTarget, it is our object and we are stay near by him.
We will upgrade elements set by functions SetUpgradeWeapons, SetUpgradeChasis i SetUpgradeShield.

IsUpgrading, SetUpgradeWeapons, SetUpgradeChasis, SetUpgradeShield
