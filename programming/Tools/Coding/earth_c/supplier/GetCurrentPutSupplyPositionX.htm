int GetCurrentPutSupplyPositionX()

Scope: private

Parameters
none

Return values
X-coordinate of point we have to fly to.

Description
This function calculates point where we have to fly to put ammo supply to current object. This function stores this point in static variable so must be called before functions GetCurrentPutSupplyPositionY and GetCurrentPutSupplyPositionZ.
