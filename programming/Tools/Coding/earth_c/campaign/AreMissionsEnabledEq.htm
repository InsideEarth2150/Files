int AreMissionsEnabledEq(int nFlagsAndValue, int nFlagsEqValue)

Scope: private

Parameters
nFlagsAndValue - "and value"
nFlagsEqValue - "eq value"

Return values
true or false

Description
It returns true if there are any enabled missions which have flags selected by nFlagsAndValue equal to nFlagsEqValue.
