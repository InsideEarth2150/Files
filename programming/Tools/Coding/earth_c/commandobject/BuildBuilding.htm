object - commandobject 

int BuildBuilding(int nX, int nY, int nZ, int nAlpha, int nBuildingID)

Parameters
nX - X-coordinate of building.
nY - Y-coordinate of building.
nZ - Z-coordinate of building.
nAlpha - building alpha angle (0-3).
nBuildingID - building identifier.

Description
Command for Builder to construct building with identifier nBuildingID in point (nX, nY, nZ) at angle nAlpha.
