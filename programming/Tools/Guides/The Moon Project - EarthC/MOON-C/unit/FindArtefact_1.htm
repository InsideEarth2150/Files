int FindArtefact(int nArtefactMask, int nFindRange)

Scope: private

Parameters
nArtefactMask - types of searched artefacts (see enum in unit)
nFindRange - range of search

Return values
True if artefact was found, false otherwise.

Description
[MP]
This function is used to search for artefacts placed in range nFindRange.
