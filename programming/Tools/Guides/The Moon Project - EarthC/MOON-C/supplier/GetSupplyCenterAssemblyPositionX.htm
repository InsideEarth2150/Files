int GetSupplyCenterAssemblyPositionX()

Scope: private

Parameters
none

Return values
X-coordinate of "Assembly" point of building we are attached to.

Description
This function returns "Assembly" point of building where we have to move after realizing job. This function calculates this point and stores it in static variable so must be called before functions GetSupplyCenterAssemblyPositionY and GetSupplyCenterAssemblyPositionZ.

See also
GetSupplyCenterAssemblyPositionY, GetSupplyCenterAssemblyPositionZ
