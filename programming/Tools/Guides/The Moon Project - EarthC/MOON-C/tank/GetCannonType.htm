object - commandobject - unit - tank

int GetCannonType(int nCannonNum)

Scope: private

Parameters
nCannonNum - cannon number.

Return values
Type of cannon number nCannonNum attached on tank:
cannonTypeLaser - laser,
cannonTypeLaserSDI - SDI (on buildings only),
cannonTypeElectric - electric cannon,
cannonTypeMachine - machine gun,
cannonTypeCannon - cannon,
cannonTypeRocket - rocket launcheer,
cannonTypeAutoRocket - rocket launcher with self-guided missiles,
cannonTypeBallisticRocket - ballistic rocket,
cannonTypeSonic - sonic cannon,
cannonTypeAntiRocket - anti-rocket,
cannonTypeDropBomb - Grenade launcher,
cannonTypeBomb - air-bombs,
cannonTypeIon - ion cannon,
cannonTypePlasma - plasma cannon,
cannonTypeSuperPlasma - plasma cannon building.
