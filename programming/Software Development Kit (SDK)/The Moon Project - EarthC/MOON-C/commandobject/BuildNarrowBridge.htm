object - commandobject 

int BuildNarrowBridge(int nX1, int nY1, int nZ1, int nX2, int nY2, int nZ2)

Parameters
nX1 - X-coordinate of bridge begin.
nY1 - Y-coordinate of bridge begin.
nZ1 - Z-coordinate of bridge begin.
nX2 - X-coordinate of bridge end.
nY2 - Y-coordinate of bridge end.
nZ2 - Z-coordinate of bridge end.

Description
Command for Builder to build narrow bridge from point (nX1, nY1, nZ1) to (nX2, nY2, nZ2).
