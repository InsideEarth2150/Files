object - commandobject 

int BuildWall(int nX1, int nY1, int nZ1, int nX2, int nY2, int nZ2)

Parameters
nX1 - X-coordinate of wall begin.
nY1 - Y-coordinate of wall begin.
nZ1 - Z-coordinate of wall begin.
nX2 - X-coordinate of wall end.
nY2 - Y-coordinate of wall end.
nZ2 - Z-coordinate of wall end.

Description
Command for Builder to build wall from point (nX1, nY1, nZ1) to (nX2, nY2, nZ2).
