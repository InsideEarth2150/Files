object - commandobject 

int TransporterSetTransportMode(int nMode)

Parameters
nMode - transporting mode:
-1 - change to next mode,
0 - change to first mode.

Description
[MP]
Command for Transporter to change transporting mode to "manual" or "automatic".
In "manual" mode Transporter is transporting only units ordered by user, in "automatic" mode is transporting our units from "auto get point" to "auto put point".
