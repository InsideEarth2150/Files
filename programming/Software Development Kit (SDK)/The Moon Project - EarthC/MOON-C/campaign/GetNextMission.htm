object - objectX - campaign 

int GetNextMission(int nMissionNum, int nNextPosition)

Scope: private

Parameters
nMissionNum - mission number
nNextPosition - next mission number.

Return values
mission number

Description
It returns one of 4 mission numbers defined when mission was registred. Each mission has up to four so called "next" missions which can be activated or desactivated base on the main mission result.
