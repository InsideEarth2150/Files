void EnableMissionsEq(int nFlagsAndValue, int nFlagsEqValue, int nEnable)

Scope: private

Parameters
nFlagsAndValue - and flag value
nFlagsEqValue - flag value used for comparision
nEnable - true or false

Return values
none

Description
This function is a bit complex and was added to simplify scripts. It enables (or disables) all missions which flags field anded with nFlagsAndValue is equal to nFlagsEqValue.
