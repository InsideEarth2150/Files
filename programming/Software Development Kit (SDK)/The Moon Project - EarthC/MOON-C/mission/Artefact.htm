object - objectX - mission 

int Artefact(int nArtefactNum, player pPlayerOnArtefact)

Scope: public

Parameters
nArtefactNum - unique artefact number
pPlayerOnArtefact - reference to player who gained artefact

Return values
true or false

Description
Event Artefact is sent whenever units stops on artefact. If this event returns true the artefact will be removed.
