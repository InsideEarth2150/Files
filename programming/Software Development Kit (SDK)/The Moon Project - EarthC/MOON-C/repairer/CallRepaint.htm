void CallRepaint(unit uTarget)

Scope: private

Parameters
uTarget - object which we have to repaint.

Return values
none

Description
This function begins repainting of object uTarget, only if object uTarget is our or in alliance and we are stay near by him.
Object is repainted to side color set by function SetRepaintSideColor.

See also
IsRepainting, SetRepaintSideColor
