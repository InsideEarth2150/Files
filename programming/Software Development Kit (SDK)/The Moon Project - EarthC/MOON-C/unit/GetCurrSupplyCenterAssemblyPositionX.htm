object - commandobject - unit 

int GetCurrSupplyCenterAssemblyPositionX()

Scope: private

Parameters
none

Return values
X-coordinate of Supply Center's assembly point.

Description
This function for flying units gets assembly point for Supply Center Building under unit. Function calculates this point and stores in static variable so should be called before functions GetCurrSupplyCenterAssemblyPositionY and GetCurrSupplyCenterAssemblyPositionZ.

See also
GetCurrSupplyCenterAssemblyPositionY, GetCurrSupplyCenterAssemblyPositionZ
