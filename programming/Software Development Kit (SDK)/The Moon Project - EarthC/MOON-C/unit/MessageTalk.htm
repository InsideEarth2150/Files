void MessageTalk(int nMessageNum)

Scope: private

Parameters
nMessageNum - number of message to play:
0 - selected talk,
1 - move command apply talk,
2 - attack command apply talk,
3 - other command apply talk,
4 - found enemy talk,
5 - help talk.

Return values
none

Description
Plays sound message number nMessageNum.
