object - commandobject - unit 

int OnTransportedToNewWorld()

Scope: public

Parameters
true - if script handled event.
false - if script dosn't handled event.

Return values
true - if script handled event.
false - if script dosn't handled event.

Description
This event is called after unit move out from units transporter.
In this event you MUST free (set to null) all pointers stored in member variables.
Returned value isn't important.
